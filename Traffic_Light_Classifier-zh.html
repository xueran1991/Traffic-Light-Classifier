2. 预处理数据¶

每个图像中加载完成后，你必须使输入和输出标准化！

输入¶

这意味着每个输入图像都应采用相同的格式，大小相同等。我们将通过对每个图像执行相同的分析来创建特征，对于这样的分类任务，重要的是类似的图像创建类似的特征！

输出¶

我们还需要将输出结果作为易于阅读且易于与其他标签进行比较的标签。将“红灯”和“绿灯”等分类数据转换为数字数据，这是一种不错的做法。

一个非常常见的分类输出是一维列表，它是指分类数的长度：在红灯、黄灯和绿灯的情况下为3； 值为0或1表示某个图像是某一类。例如，由于我们有三个类（红灯、黄灯和绿灯），则可以按顺序列出一个列表：[红灯值，黄灯值，绿灯值]。一般来说，顺序无关紧要，在这种情况下，我们选择的顺序是[红灯值，黄灯值，绿灯值]，便于反映垂直顺序中每个灯的位置。

红灯的标签应该是：[1,0,0]，黄灯应该是：[0，1，0]，绿灯应该是：[0，0，1]。这些标签被称为独热编码标签。

（注：使用 机器学习算法 时，独热编码将尤为重要）。

def standardize(image_list):
    
    # Empty image data array
    standard_list = []

    # Iterate through all the image-label pairs
    for item in image_list:
        image = item[0]
        label = item[1]

        # Standardize the image
        standardized_im = standardize_input(image)

        # One-hot encode the label
        one_hot_label = one_hot_encode(label)    

        # Append the image, and it's one hot encoded label to the full, processed list of image data 
        standard_list.append((standardized_im, one_hot_label))
        
    return standard_list

# Standardize all training images
STANDARDIZED_LIST = standardize(IMAGE_LIST)

## 将图像进行掩码处理， 使输出图像只保留亮度值较高的部分
def get_masked_image(image, lower_color, higher_color):
    lower_color = np.array(lower_color)
    higher_color = np.array(higher_color)
    mask = cv2.inRange(image, lower_color, higher_color)
    return cv2.bitwise_and(image, image, mask=mask)

## 通过观察图像发现，红绿灯图像大都在图像中呈现竖直方向上居中，
## 水平方向上三种颜色的灯呈现上中下3部分排列
## 这里去除红绿灯光源两侧和上下方向上的部分背景
def get_croped_image(image, row_crop=5, col_crop=10):
    return image[row_crop:-row_crop, col_crop:-col_crop, :]

（问题1）：你创建的特征如何帮你区分这3类交通信号灯图像？¶

答案： 我创建的特征是图像中亮度最高的行在图像中的位置。

将你的分类算法（也称为“模型”）的输出与真实标签进行比较并确定准确度。

# Constructs a list of misclassified images given a list of test images and their labels
# This will throw an AssertionError if labels are not standardized (one-hot encoded)

def get_misclassified_images(test_images):
    # Track misclassified images by placing them into a list
    misclassified_images_labels = []

    # Iterate through all the test images
    # Classify each image and compare to the true label
    for image in test_images:

        # Get true data
        im = image[0]
        true_label = image[1]
        assert(len(true_label) == 3), "The true_label is not the expected length (3)."

        # Get predicted label from your classifier
        predicted_label = estimate_label(im)
        assert(len(predicted_label) == 3), "The predicted_label is not the expected length (3)."

        # Compare true and predicted labels 
        if(predicted_label != true_label):
            # If these labels are not equal, the image has been misclassified
            misclassified_images_labels.append((im, predicted_label, true_label))
            
    # Return the list of misclassified [image, predicted_label, true_label] values
    return misclassified_images_labels

（问题2）：在将这些误分类可视化之后，你认为你的分类算法存在哪些缺陷？请至少发现两个缺陷。¶

答案：

  1. 算法只利用了等绿灯中不同颜色灯的位置信息，而没有利用其颜色信息。当位置信息无法表达红绿灯的颜色时，检测就会失败，如上图所示的4个错误分类。

  2. 为消除红绿灯背景干扰，图像进行了切割，这提升了算法在整个数据集上的表现，但对于某些图像，也会丢失重要的位置信息。
